# Aggregate function

Use the `SUM` function to get the total duration of all films:
```sql
select sum(duration)
from films;
```

Get the duration of the longest film:
```sql
select max(duration)
from films;
```

Get the average duration of all films:
```sql
select avg(duration)
from films;
```

Get the average amount grossed by all films:
```sql
select avg(gross)
from films;
```
